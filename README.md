# Bayan Sulu Kids

Bayan Sulu Kids is an interactive, educational mobile game for children, themed around the Bayan Sulu confectionery brand and Kazakh culture. The adventure features KamBot, a friendly camel who guides children through various mini-games spanning across Kazakhstan's iconic locations.

## Что мы сделали (Последние обновления)

- **Интеграция с DeepSeek API**: Перевели чат с маскотом Ботой на модель `deepseek-v4-flash`. Отключили режим размышления (thinking mode) для моментальных ответов.
- **Обновление дизайна**: Полностью переработали верхнюю панель управления (Top Bar) и список игр на карте. Сделали список горизонтальным, добавили индикатор уровня и красивый стеклянный стиль (Glassmorphism).
- **Бесплатная озвучка (TTS)**: Перевели озвучку с платного ElevenLabs на бесплатный стандартный пакет `flutter_tts`, который работает без ключей и интернета.
- **Лэндинг и репозитории**: Создали отдельный сайт-визитку для скачивания APK и залили код в отдельные репозитории на GitHub.

## Инструкция по запуску

Для запуска приложения вам понадобится установленный Flutter SDK.

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/Eraly-ml/bota-app.git
   cd bota-app
   ```
2. **Настройте переменные окружения**:
   Создайте файл `.env` в корневой папке проекта и добавьте туда ваш ключ от DeepSeek:
   ```env
   DEEPSEEK_API_KEY=ваш_ключ_от_deepseek
   ```
3. **Установите зависимости**:
   ```bash
   flutter pub get
   ```
4. **Запустите приложение**:
   ```bash
   flutter run
   ```
   Или соберите релизный APK:
   ```bash
   flutter build apk --release
   ```

## Текущая архитектура

Приложение изначально планировалось с бэкендом на Supabase, но на данный момент все данные (слова, вопросы викторины) работают локально и загружаются из локальных JSON файлов (например, `assets/data/quiz.json`). `SupabaseService` в коде является заглушкой для локальных данных.

## Особенности игры

- **Интерактивная карта**: Путешествуйте по регионам Казахстана (Астана, Алматы, Чарын и др.).
- **Мини-игры**: Память, Поймай конфету, Сборка юрты, Счёт с Ботой, Викторины.
- **Система Ботакоинов**: Зарабатывайте виртуальную валюту за игры и ежедневный вход.
- **Родительский контроль**: Безопасный раздел (ПИН-код по умолчанию: 1234) для контроля времени и просмотра статистики ребенка.
- **Двуязыковая поддержка**: Полная локализация на казахский и русский языки.
